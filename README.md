# Advent of Code 2021

https://adventofcode.com/

## Stack
- Node 16
- pnpm (npm compatible)
- Typescript
- Jest 27
- Esbuild

## Install 
```shell
pnpm i
```

## Run
```shell
pnpm run test
```

## Build
For a few days, I have some visualization in the console.
```shell
pnpm run build
node dist/day10/SyntaxScoring.js 
node dist/day11/DumboOctopus.js
```
